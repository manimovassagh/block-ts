
# Block-TS

Block-TS is a TypeScript project designed to demonstrate best practices and efficient coding techniques.

## Installation

To install the project dependencies, run:

```bash
npm install
```

## Usage

To build the project, run:

```bash
npm run build
```

To start the project, run:

```bash
npm start
```
